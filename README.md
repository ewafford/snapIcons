## Usage

1. Include jQuery:

	```html
	<script src="http://ajax.googleapis.com/ajax/libs/jquery/2.1.4/jquery.min.js"></script>
	```

2. Include plugin's code:

	```html
	<script src="dist/jquery.snapicons.min.js"></script>
	```

3. Call the plugin:

	```javascript
	$("#element").snapIcons();
	```
